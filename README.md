# Amazon Book List Analyzer

## Overview

This is a small project for processing an online dataset from Kaggle with a Amazon Book List focused in listing:

- The 20 books with most reviews
- The top 10 authors with the most books
- The top 25 books with best ratings
- The top 10 authors with best ratings

### Input
- `bestsellers with categories.csv`


### Technologies
 - Python 3
 - Pandas

### Data Source

[bestsellers with categories.csv](https://www.kaggle.com/datasets/obaidhere/amazon-bestsellers-price-vs-user-rating-analysis)
